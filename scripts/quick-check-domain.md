# Quick Check Domain Info

## Cách Nhanh Nhất

1. **Mở browser và truy cập:**
   ```
   https://whois.net/gaigu1.net
   ```

2. **Hoặc:**
   ```
   https://www.whois.com/whois/gaigu1.net
   ```

3. **Xem thông tin:**
   - **Creation Date** = Ngày đăng ký
   - **Expiration Date** = Ngày hết hạn
   - **Registrar** = Nhà đăng ký

## Tính Thời Gian Đã Thuê

Sau khi có Creation Date, tính:
- Số ngày = (Hôm nay - Creation Date)
- Số năm = Số ngày / 365.25
- Số tháng = Số ngày / 30.44

## Ví Dụ

Nếu Creation Date là `2023-01-15`:
- Hôm nay: `2024-12-06`
- Đã thuê: ~691 ngày = ~1.89 năm = ~22.7 tháng
